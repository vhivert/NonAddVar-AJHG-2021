PhenoCpp Simulator is a C++ software. The software works under Linux and Mac. However, because it uses the openmp library, some personalized changes 
may be required in the Makefile to compile on Mac. 

How to install the software? Using a terminal, change directory to the software repository (where the Makefile is located) and type 'make clean all' to install the software

How to use the software? This is a command line software, the binary file is located in the bin sub-directory. To obtain help type : PathToTheBinRepository/PhenoSim --help
